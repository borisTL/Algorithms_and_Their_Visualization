# Algorithm Visualization

An interactive platform to visualize a broad spectrum of algorithms, assisting in understanding their inner workings.

## Featured Algorithms:

### Search Algorithms:
- Binary Search
- Breadth-First Search (BFS)
- Depth-First Search (DFS)

### Sorting Algorithms:
- Merge Sort
- Quick Sort
- Insertion Sort
- Selection Sort
- Counting Sort
- Heap Sort

### Graph Algorithms:
- Kruskal's Algorithm
- Floyd Warshall
- Dijkstra's Algorithm
- Bellman Ford
- Floyd's Cycle Detection
- Topological Sort
- Kahn's Topological Sort

### Others:
- Kadane's Algorithm
- Lee's Algorithm
- Flood Fill
- Union Search
- KMP Algorithm
- Huffman Coding
- Quick Select
- Boyer-Moore Majority Voting
- Euclidean Algorithm
